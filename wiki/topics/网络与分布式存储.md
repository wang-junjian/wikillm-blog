## 关键知识点

- **负载均衡**：HAProxy 核心配置（global/defaults/frontend/backend）、Docker 容器化部署
- **NFS**：服务端导出配置、客户端挂载、权限管理
- **MinIO**：独立模式、分布式模式、Docker Compose 部署、mc 客户端命令、S3 兼容 API
- **Verdaccio**：私有 npm 注册表部署、config.yaml 配置、自动缓存上游包、用户认证管理
- **容器网络**：Docker 网络模型、桥接/覆盖网络
- **存储协议**：对象存储（S3）、文件存储（NFS）、块存储对比
