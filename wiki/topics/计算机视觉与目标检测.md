# 计算机视觉与目标检测

> 领域：大模型与算法研究

## 概述

计算机视觉是人工智能的"眼睛"，负责从图像和视频中提取语义信息。本主题聚合从 YOLO 系列目标检测、OpenVINO 模型优化与推理部署，到 Ultralytics 生态工具链的完整实践，覆盖训练、转换、部署、性能分析全流程。

技术栈涵盖三条部署路径：通用 GPU（Ultralytics YOLO）、Intel 硬件（OpenVINO IR 格式）、边缘设备（ONNX Runtime + TensorRT/CoreML）。YOLO 系列从 v3（Darknet 原生）到 v5（PyTorch 工程化）再到 v8（统一 CLI），展现了目标检测从研究到工业的演进。2024-2025 年，视觉语言模型（CogVLM2、Phi-3-vision、GLM-4V-9B、DeepSeek-OCR）将 OCR、图像理解、文档分析整合到统一框架，形成对传统 pipeline 的降维打击。

## 关键知识点

1. **YOLO 系列演进** — v3（Darknet 原生训练、PASCAL VOC → YOLO 格式转换）、v4（CSPDarknet53 + SPP + PANet）、v5（PyTorch 工程化、data.yaml 配置）、v8（统一 yolo CLI、13 种格式导出）
2. **推理格式权衡** — PyTorch vs ONNX vs OpenVINO vs TensorRT 速度对比（YOLOv8 场景 PyTorch 比 ONNX 快 31-38%）、模型体积变化（训练 24MB → 推理 6MB → ONNX 12MB）
3. **Ultralytics 生态** — yolo CLI（TASK MODE ARGS）、Hub 云端训练、Roboflow 数据标注/版本管理/SDK 部署、Comet/ClearML 可视化
4. **FastAPI 推理服务** — ONNX Runtime + Gunicorn 多进程、Docker buildx 多架构（amd64/arm64）、Swagger UI 内网离线部署
5. **OpenVINO 优化** — 五步工作流（计划→训练→转换→调整→部署）、IR 格式（.xml + .bin）、Model Optimizer 转换、PrePostProcessor 嵌入预处理
6. **性能分析** — benchmark_app（同步 LATENCY vs 异步 THROUGHPUT）、逐层 Profiling（PERF_COUNT）、Cross Check Tool（跨设备/跨精度精度对比）
7. **OCR 引擎** — EasyOCR vs PaddleOCR vs Tesseract 基准测试（CPU/CUDA 速度、显存占用、PyTorch 版本绑定问题）
8. **多模态 VLM** — CogVLM2（19B、OCR/表格/信息提取）、Phi-3-vision（4B、128K 上下文）、GLM-4V-9B（中英双语）、DeepSeek-OCR（上下文光学压缩、SAM+CLIP+压缩器）
9. **数据格式转换** — PASCAL VOC（XML 绝对坐标）→ YOLO（归一化中心坐标）公式、LabelImg 标注、数据集 zip 打包
10. **边缘视觉应用** — Reachy Mini 机器人（YOLO/MediaPipe 头部追踪、线程安全帧缓冲、SmolVLM2 本地部署）
