# 大型语言模型

大型语言模型（Large Language Model, LLM）是基于海量文本数据训练的深度学习模型，能够理解、生成和操纵自然语言文本。现代 LLM 通过 Transformer 架构和规模化的预训练（Pre-training），在数千亿甚至万亿级词汇的语料上学习语言的统计规律与知识表征，从而具备了强大的文本理解、逻辑推理、代码生成、多轮对话和跨语言能力。LLM 已成为当代人工智能应用的核心基础设施，深刻改变着软件开发、知识管理、内容创作、科学研究等多个领域。

从 2018 年 GPT 系列开创的"预训练+微调"范式，到 2020 年 GPT-3 展现的 Few-shot 学习能力，再到 2022 年 ChatGPT 引爆的对话式 AI 革命，LLM 经历了从实验室到产业化的快速演进。2024-2026 年间，Claude、Gemini、Llama、Qwen、DeepSeek 等多极模型格局形成，模型能力持续攀升的同时，推理成本大幅降低，开源生态蓬勃发展。LLM 不仅是技术工具，更成为知识工作者的"认知外骨骼"。

## 核心架构：Transformer

LLM 的核心架构是 Transformer（Vaswani et al., 2017），其关键创新在于自注意力机制（Self-Attention）。Transformer 通过多头注意力层让模型在处理每个词时关注输入序列的所有位置，捕捉长距离依赖关系。相比传统的 RNN 和 LSTM，Transformer 天然支持并行计算，使大规模训练成为可能。

现代 LLM 普遍采用 Decoder-only 架构（如 GPT 系列、Llama、DeepSeek），通过自回归方式逐词生成文本。关键组件包括：位置编码（Positional Encoding）为序列引入顺序信息；层归一化（Layer Normalization）稳定训练过程；前馈网络（Feed-Forward Network）提供非线性变换能力；残差连接（Residual Connection）缓解深层网络的梯度消失问题。模型的参数量从最初的数亿扩展到数万亿，上下文窗口从数千扩展到数百万 tokens。

## 训练方法

LLM 的训练通常分为三个阶段：预训练、微调和对齐。

**预训练**（Pre-training）是 LLM 获取基础能力的关键阶段，通过在海量无标注文本上执行下一个词预测（Next Token Prediction）任务，模型学习到语言的语法、语义和世界知识。预训练消耗大量计算资源，通常需要在数千块 GPU 上运行数周至数月。预训练数据涵盖网页、书籍、论文、代码、对话等多种语料，数据质量和多样性直接决定模型上限。

**微调**（Fine-tuning）在预训练模型基础上，使用特定领域或任务的有标注数据进行二次训练，使模型适配具体应用场景。参数高效微调方法如 LoRA、QLoRA、Adapter 等大幅降低了微调成本，使开发者能在消费级硬件上完成领域适配。

**对齐**（Alignment）通过人类反馈强化学习（RLHF）或直接偏好优化（DPO），使模型的输出符合人类价值观和偏好，降低有害输出的概率，提升有用性和安全性。对齐是 LLM 从"能说话"到"说好话"的关键步骤。

## 涌现能力与规模定律

LLM 最引人注目的特征之一是涌现能力（Emergent Abilities）——当模型规模超过某个阈值时，会展现出训练时未明确设计的能力，如多步推理、代码生成、数学解题、工具使用等。这种现象与规模定律（Scaling Laws）密切相关：模型能力随参数量、数据量和计算量的增加而可预测地提升。

Chinchilla 定律指出，在计算预算固定的情况下，模型参数量和训练数据量应按 1:1 比例扩展。这推动了训练策略从"堆参数"转向"参数-数据平衡"。DeepSeek-V3 等新一代模型通过 MoE（Mixture of Experts）架构，在保持推理计算量不变的前提下扩展总参数规模，实现了性能与效率的双重突破。

## 应用场景

LLM 的应用已渗透到各行各业：**智能对话与客服**提供 24/7 多语言交互服务；**代码生成与辅助编程**（如 GitHub Copilot、Claude Code）大幅提升开发效率；**知识管理与信息检索**通过 RAG 架构实现企业级知识问答；**内容创作与翻译**支持多模态内容生成；**科学研究**辅助文献综述、假设生成和数据分析；**教育**提供个性化辅导和自动评测。
