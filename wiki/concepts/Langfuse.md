# Langfuse

Langfuse 是一个开源的 LLM 工程可观测性平台，提供追踪、监控、评估和调试 LLM 应用的能力。它支持记录 LLM 调用的完整链路，包括输入输出、延迟、成本和 token 使用量。

Langfuse 帮助开发者更好地理解和优化 LLM 应用的性能，是 LLM 应用开发和运维的重要工具。
