# GitHub Copilot

GitHub Copilot 是 GitHub 推出的 AI 编码助手，基于 OpenAI 的模型构建，集成在 VS Code、JetBrains 等主流 IDE 中。它提供实时代码补全、函数生成、测试编写等功能，是目前最广泛使用的 AI 编码工具之一。

Copilot 支持多种编程语言，并逐步扩展到 Copilot Chat、Copilot Workspace 等更全面的开发辅助功能。
