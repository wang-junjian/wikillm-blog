# Ollama

Ollama 是一个轻量级的本地大语言模型运行框架，支持在个人设备上快速下载、管理和推理开源 LLM（如 LLaMA、Mistral、Qwen 等）。它提供了简洁的命令行接口和 REST API，使开发者和研究者能够在本地环境中高效地部署和测试各类开源大模型，是本地 LLM 生态的重要基础设施。
