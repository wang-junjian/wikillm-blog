# LiteLLM

LiteLLM 是一个统一的 LLM API 代理服务，将不同厂商（OpenAI、Anthropic、Google、Azure 等）的 API 调用统一为 OpenAI 兼容格式。它提供了负载均衡、故障转移、成本追踪和速率限制等功能，帮助开发者在多模型、多供应商环境中简化集成和运维工作。
