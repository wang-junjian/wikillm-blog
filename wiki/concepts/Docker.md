# Docker

Docker 是一个开源的容器化平台，通过操作系统级别的虚拟化技术，将应用及其依赖打包成轻量级、可移植的容器。Docker 解决了"在我机器上能跑"的环境一致性问题，已成为现代软件开发和部署的标准基础设施。

Docker 容器相比传统虚拟机更轻量、启动更快，配合 Docker Compose、Kubernetes 等编排工具，可以实现从开发到生产的全链路容器化部署。
