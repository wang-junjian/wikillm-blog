# FastMCP

FastMCP 是一个用于快速构建 MCP 服务器的 Python 开发框架，提供了简洁的 API 来定义工具、资源和提示。它简化了 MCP 协议的开发流程，使开发者能够快速将现有功能暴露给 AI 代理使用。

FastMCP 是 MCP 生态中流行的服务器端实现框架之一，支持同步和异步工具定义。
