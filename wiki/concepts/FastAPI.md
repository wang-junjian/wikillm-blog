# FastAPI

FastAPI 是一个现代、高性能的 Python Web 框架，基于 Starlette 和 Pydantic 构建，支持异步编程和自动 API 文档生成。它以出色的开发效率和运行时性能著称，广泛用于构建 LLM 推理服务、RESTful API 和微服务，是 Python 生态中最受欢迎的 Web 框架之一。
