# Kilo Code

Kilo Code 是一款开源的 AI 编码助手，可作为 VS Code 扩展使用。它支持多种 LLM 提供商，提供代码生成、调试辅助、项目规划等功能，是开源编码助手生态中的重要一员。

Kilo Code 注重灵活性和可扩展性，允许开发者根据需求配置不同的 AI 模型和工具链。
