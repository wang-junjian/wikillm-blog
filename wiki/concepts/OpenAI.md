# OpenAI

OpenAI 是全球领先的人工智能研究公司，以 GPT 系列大模型闻名。其产品包括 GPT-4、GPT-4o 等大语言模型，以及 DALL·E 图像生成、Sora 视频生成等。OpenAI 通过 API 服务和 ChatGPT 产品，推动了生成式 AI 的普及与商业化。

OpenAI 在模型能力、安全研究和商业化方面处于行业前沿，是与 Anthropic、Google DeepMind、Meta 并列的大模型核心厂商之一。
