# MLX

MLX 是 Apple 开发的专为 Apple Silicon 芯片优化的机器学习框架，支持模型训练、微调和推理。它提供了类似 NumPy 和 PyTorch 的 API 设计，能够高效利用 Apple 芯片的统一内存架构。

MLX 特别适合在 Mac 设备上进行 LLM 微调和本地推理，是 Apple 生态中重要的机器学习基础设施。
