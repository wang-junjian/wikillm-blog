# FastChat

FastChat 是 LMSYS（Large Model Systems Organization）开源的 LLM 服务与评估平台，支持多模型部署、对话生成和模型评测。它是 Vicuna 等开源聊天模型的核心推理后端，提供了兼容 OpenAI 的 API 接口和分布式多模型部署能力，是开源 LLM 部署生态的重要组件。
