# Continue

Continue 是一款开源的 AI 编码助手，可集成到 VS Code 和 JetBrains 等 IDE 中。它支持多种 LLM 后端，提供代码补全、代码库问答、重构建议等功能，帮助开发者在熟悉的开发环境中使用 AI 辅助编程。

Continue 强调可定制性和隐私保护，允许开发者自由选择模型提供商和配置方式。
