# Claude

Claude 是 Anthropic 开发的大语言模型系列，以其安全性、长上下文窗口和强大的推理能力著称。Claude 系列包括多个版本（如 Claude 3.5 Sonnet、Claude 3 Opus、Claude 4 等），支持文本、图像等多模态输入。

Claude 在编程、分析、创作等任务上表现出色，是目前最主流的商用 LLM 之一。
