# VS-Code

Visual Studio Code（VS Code）是微软开发的免费、开源代码编辑器，凭借其丰富的扩展生态和内置 AI 辅助功能（如 GitHub Copilot）成为开发者最常用的 IDE 之一。它支持多种编程语言、调试工具、Git 集成和远程开发，是现代软件开发工作流的核心工具。
