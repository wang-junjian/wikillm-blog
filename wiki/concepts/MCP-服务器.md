# MCP 服务器

MCP 服务器是基于 Model Context Protocol（MCP）协议运行的服务端组件，负责向 AI 代理暴露工具、资源和能力。MCP 服务器遵循标准化的 JSON-RPC 协议，使 LLM 能够以统一方式调用外部 API、数据库、文件系统和其他服务。

MCP 服务器是 MCP 生态的核心基础设施，常见的实现包括官方参考实现和社区开发的各类适配器。
