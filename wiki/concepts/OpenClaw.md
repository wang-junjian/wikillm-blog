# OpenClaw

OpenClaw 是一个开源的 AI Agent 编排框架，专注于构建可组合、可扩展的多智能体系统。它提供了一套标准化的工具调用、任务分解和 Agent 协作机制，使开发者能够构建处理复杂任务的 AI 应用。

OpenClaw 强调模块化设计和社区驱动的工具生态，支持多种 LLM 后端，是开源 AI Agent 基础设施中的重要项目之一。
